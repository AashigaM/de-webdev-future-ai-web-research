# Dual Enrollment: AI Research & Digital Showcase

**Live Demo:** https://aashigam.github.io/de-webdev-future-ai-web-research/#top

## Project Overview

This project was built for my Dual Enrollment (DE) Web Page Development course as a research-backed digital showcase titled "Future AI: Exploring Artificial Intelligence." It synthesizes core subfields of modern computer science—including Machine Learning, Natural Language Processing (NLP), and Computer Vision—alongside real-world case studies in bio-engineering and autonomous systems. The primary technical focus was on executing advanced CSS multi-column layouts, card grid systems, visual hierarchy, and integrating cited academic research into a clean web interface.

## Key Technical Skills Demonstrated

* **Advanced CSS Layouts & Card Systems:** Structured complex multi-column responsive grid layouts and custom media card styling using advanced CSS spacing, padding, custom borders, and flex structures.
* **Academic Content Integration & Citations:** Formatted research content featuring a dedicated Works Cited section with academic references (including MIT Press and foundational AI literature) and linked external source attributions.
* **Semantic Web Architecture:** Implemented accessible HTML5 sectioning and inline jump-link navigation (`#categories`, `#case-study`, `#citations`) to optimize UX when reading detailed technical research.
